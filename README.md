### Spherical sky for Map creators
Place `env_sky.fgd` to `Garrysmod/bin/`. Then goto Hammer++, `Tools` -> `Options` -> `Game Data files:` -> `Add` `env_sky.fgd`. 
<img width="1222" height="543" alt="image" src="https://github.com/user-attachments/assets/385f8356-c695-42c1-8278-59b6ab372854" />

Params of `env_sky`:
- Static Sky Textue (`staticskytexpath`) — Set static sky texture which will be shown if static mode enabled.
- Sky Brightness Scale (`brightness`) — Defines how bright the sky is.
